# Faz 1 — UI/UX ve Sistem Tasarımı

**Tahmini süre:** 1–2 hafta  
**Durum:** Bekliyor (Faz 0 tamamlanmadan başlanmaz)

## Hedef

CMS panelinin bilgi mimarisini ve görsel tasarım temelini oluşturmak; Gusto Kartepe web sitesi sayfa yapısını belirlemek.

## Yapılacaklar

- [ ] CMS panel bilgi mimarisi (IA) çıkarmak
- [ ] Ana ekran wireframe'leri (Dashboard, Sayfa Yönetimi, Medya, SEO)
- [ ] Design system temeli — Tailwind token yapısı (`--bg-base`, `--brand-primary`, vb.)
- [ ] Light / dark mode tasarım yaklaşımını belgelemek
- [ ] Component listesini finalize etmek
- [ ] Gusto Kartepe web sitesi sayfa yapısını onaylatmak (7 sayfa)
- [ ] Sidebar, Topbar, Form, Tablo, Modal bileşen spec'lerini DESIGN.md'ye eklemek

## Tamamlanma Kriteri

- Tüm MVP ekranları için en az low-fidelity wireframe mevcut
- Tailwind token dosyası (`tokens.css` veya `tailwind.config`) taslak olarak hazır
- Light/dark mod geçiş stratejisi kararlaştırılmış
- Component listesi onaylanmış

## Bağlantılı Dosyalar

- [DESIGN.md](./DESIGN.md) — renk tokenları, tipografi, bileşen spec'leri
- [CONTENT.md](./CONTENT.md) — sayfa içerikleri (wireframe'lerde kullanmak için)
- [FAZ-2.md](./FAZ-2.md) — bu fazda alınan kararlar backend modelini etkiler
